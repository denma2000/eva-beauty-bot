# Eva Beauty Bot

Telegram bot for nail salon appointment management.

## Setup

1. Clone the repository
2. Install dependencies:
```npm install```
3. Create `.env` file from `.env.example` and add your Telegram bot token
4. Run the bot:
```npm start```

## Features

- Service management
- Appointment booking
- Schedule management
- Client notifications
